Visual GUI Editor

Edit every inventory slot, title, filler material, and item position directly in-game or via the web editor — zero YAML knowledge needed.

NEW
⚡

Great Performance

Every heavy task runs asynchronously — database I/O, leaderboard updates, boost history, Discord webhooks — so your TPS stays rock-solid.

CORE
🎒

Booster Inventory

Each player holds their own collection of boosters — personal, team, or global — stored safely across server restarts.

CORE
👥

3 Boost Scopes

Global boosters benefit everyone. Personal boosters are private. Team boosters integrate with DonutTeams — one system, all scopes.

CORE
🔗

Discord Webhooks

Automatically post rich embeds to your Discord server whenever a boost starts or ends. Keeps your community looped in 24/7.

API
🛠

Developer API

Full Java API with cancellable events, boost multiplier overrides, custom types, and PlaceholderAPI integration for HUDs and scoreboards.

SCHEDULER
🤖

Smart AI Scheduler

Learns your server's peak hours from engagement data and suggests — or auto-schedules — boosts for maximum impact. Configurable via scheduling.yml with full cron support and a live in-game dashboard.

v1.0
🏆

Milestones & Achievements

Reward players for hitting boost activation milestones. Configurable LuckPerms hooks grant groups on achievement.

v1.0
🛡

Fraud Detection

Built-in abuse detection flags suspicious activation patterns, rate-limits exploiters, and logs events for admin review.

CORE
⬇

Negative Boosters

Apply debuff multipliers (below 1×) as a challenge or punishment mechanic — fully configurable per boost type.

Performance

    
Built for heavy servers

    
Every slow operation runs off the main thread. GUIBoost never blocks your TPS — even with thousands of players and hundreds of active boosters.


    

      

        <0.1ms
        
Average boost tick overhead on the main thread

      

      

        100%
        
Database operations run fully asynchronously

      

      

        0 leaks
        
All BossBar, scheduler, and task handles cancelled cleanly on /reload
